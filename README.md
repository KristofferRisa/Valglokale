# Source code for the web app "lokale"
this app uses the open API for valglokale and it lets you filter the data.

## getting started

<pre>
<code>
npm install -g npx
</code>
</pre>

## dev

<pre>
<code>
cd valglokale
npm run dev
</code>
</pre>

## deploy
<pre>
<code>
cd valglokale
npm run build
npm run start
nuxt generate
</code>
</pre>

## test
<pre>
<code>
cd valglokale
npm run test
</code>
</pre>
